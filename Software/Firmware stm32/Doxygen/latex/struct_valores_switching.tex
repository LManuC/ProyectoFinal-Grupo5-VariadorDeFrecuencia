\doxysubsection{Descripción detallada}
Estructura de trabajo del ISR del timer de switching. 

Contiene los parámetros que determinan en qué puntos (ticks) se disparan las comparaciones CCR para conmutar las fases y el estado de conmutación (encendido/apagado) de cada pierna. El campo \doxylink{struct_valores_switching_a6794e128f58f056963a3ed7efcf8e57a}{cuadrante} indica el sector SVM actual (0..5).
